# Solitaire iOS App

Classic Solitaire made with SwiftUI.
- Game state is persisted with UserDefaults
- Model-View-ViewModel (MVVM) architectural pattern

## Features

- Auto-move on card tap
- Responsive layout
- Score system
- Game timer
- Theme settings
